Installation

'Easy Way' - Upload store package that goes with template to your server, so you would have fully functional store in 2 minutes. 

  1. Extract files from archive and go to "Demo Store" Folder
  2. You will find 2 files, one is database(less size) and the second is magento store. You will need upload all files to your server public_html directory and import database to your database. So in the end you will have files and database installed on your server 
  3. Change database credentials in local.xml file, which is located on your serve in /app/ directory
  4. Change address setting to your URL in core_config_data "Config Data", web/unsecure/base_url & web/secure/base_url

 

'Normal Way' - Install template into your existing magento store. 

  1. Disable your magento cache & clear the store cache under var/cache and all cookies for your store domain
  2. Disable Compilation Mode (if enabled)
  3. Log out from Magento Admin panel if you are logged in
  4. Upload files from "/Magento Template/Public" directory to your magento installation
  5. Check that magento has permissions to read tempalate files.
  6. Create all static blocks, you will find them in "Magento Static Blocks" directory, each file contains content for one static block. Ex: file with name checkout_banner.txt goes with content for "checkout_banner" static block. Filename = Static block name.
  7. Insert content for your homepage from "homepage_content.txt" file which is located in CMS directory
  8. Please do not forget change Design Setting and to adjust fonts and colors in setting section as shown on next two images
